# Laravel Facility Booking Website

## Overview
This repository contains the source code for a Laravel-based facility booking website. The application allows users to reserve various facilities with ease while providing an intuitive interface for administrators to manage bookings.

## Features
- User authentication
- Facility listing
- Booking management
- Calendar integration
- Payment processing
